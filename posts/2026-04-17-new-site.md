The site is now built with [Astro](https://astro.build) instead of hand-edited HTML files.

The content is identical — the security, AI, and links pages are all still there. What changed is under the hood: pages are now Astro components instead of duplicated HTML, dark mode no longer flashes on load, and the build generates a proper sitemap.

I also added this blog. Writing is the main way personal sites get traffic, so it made sense to wire it up. Posts will be a mix of security topics, things I've been reading, and occasional notes like this one.

If you bookmarked any of the old `.html` URLs (like `/security.html`), redirects are in place and they'll forward to the new clean paths.
