Reference & Frameworks
Essential knowledge bases and frameworks for security work.
  • DataBreaches
    Community News Site
    Independent news site tracking data breaches, privacy violations, and cybersecurity incidents. Comprehensive coverage of breach notifications and regulatory responses.
  • MITRE EMB3D
    MITRE Corporation
    Knowledge base of cyber threats to embedded devices. Covers attack techniques, vulnerabilities, and mitigations specific to IoT and embedded systems.
  • MITRE ATT&CK
    MITRE Corporation
    Knowledge base of adversary tactics and techniques based on real-world observations. Essential reference for threat hunting, detection engineering, and security assessments.
  • MITRE ATLAS
    MITRE Corporation
    Framework for understanding adversarial threats to AI and machine learning systems. Covers attack techniques specific to AI/ML environments and defenses.
  • Hacking The Cloud
    Community Project
    Encyclopedia of cloud security attack techniques and tactics. Covers AWS, Azure, and GCP exploitation methods with practical examples and detection guidance.
  • Known Breaches
    GitHub Project
    Comprehensive database of historical data breaches and security incidents. Includes timelines, impact assessments, and lessons learned from major breaches.
